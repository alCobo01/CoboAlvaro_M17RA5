# Mondongo's Adventure

## Keyboard controls

| Action | Key |
|---|---|
| Move Forward | `W` |
| Move Backward | `S` |
| Move Left | `A` |
| Move Right | `D` |
| Attack | `Enter` |
| Sprint (Hold) | `Left Shift` |
| Jump | `Space` |
| Crouch (Hold) | `Ctrl` |
| Interact | `E` |
| Dance | `V` |

## Notes

- If you save the game and return to the main menu, you can use `Continue` to load your saved data.
